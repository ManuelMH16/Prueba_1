# Prueba_1 - Simulacion de Ciber Resiliencia

Basado en la noticia: **"Ciberseguridad: brechas criticas que aun desafian la proteccion de la informacion empresarial"**  
Fuente: Lima, Peru - 16 de abril de 2026

## Contexto

Segun el *Kyndryl Readiness Report 2025*, solo el **31% de las organizaciones** se siente preparada para enfrentar riesgos externos. Este proyecto simula los 6 pilares de ciber resiliencia recomendados por expertos.

## Pilares simulados

1. Visibilidad integrada de riesgos
2. Modernizacion de sistemas
3. Backup inmutable y recuperacion
4. Zero Trust en accesos e identidades
5. Inteligencia Artificial para deteccion de amenazas
6. Arquitectura resiliente

## Ejecucion

```bash
python3 ciber_resiliencia.py
```

No requiere dependencias externas.
